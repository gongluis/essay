## **上标**

---

依赖模块: pymdownx.caret

```text
10^2^
```

效果

10^2^

针对英文描述第几的上标，可以`1st`这种方式(依赖模块: pymdownx.smartsymbols)

| 用法                 | 效果              |
| :------------------ | :---------------- |
| `1st 2nd 3rd`       | 1st 2nd 3rd       |
| `1^st^ 2^nd^ 3^rd^` | 1^st^ 2^nd^ 3^rd^ |


## **下标**

---

依赖模块: pymdownx.tilde

```text
H~2~O
```

H~2~O
