## **注解标题和内容支持多种样式**

---

**例1. 标题里使用斜体**

```text
!!! note "*标题e*"
    内容
```

!!! note "*标题*"
    内容

**例2. 内容里使用h1-h6**

```text
!!! note "标题"
    # h1
    ## h2
    ### h3
```
